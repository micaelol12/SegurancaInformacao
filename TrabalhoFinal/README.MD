# Trabalho Final

Programa de criptografia  e  descriptografia  através  do  algoritmo  AES


## Como Rodar

- Para rodar execute

```
python Encrypter
```

## Como Testar

- Para testar é necessário instalar as dependência e para isso execute:

```
pip install -r requirements.txt
```

- Depois é só rodar os testes:

```
pytest
```
